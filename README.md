# WaveWarZ Restream Overlay

"Now battling" lower-third browser-source overlay for Restream/OBS.
Back Music, Not Memes.

Usage: deploy, then paste into Restream's widget URL box:

    https://<your-deploy>/?left=ARTIST+A&right=ARTIST+B&sub=Optional+line

All params optional. Transparent background, gold accent, reduced-motion safe.
